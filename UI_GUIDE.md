## Responsive Behavior

### Desktop (> 1024px)
- Sidebar: 280px fixed width
- Content: Remaining space
- Document grid: 3-4 columns

### Tablet (768px - 1024px)
- Sidebar: 280px fixed width
- Content: Remaining space
- Document grid: 2-3 columns

### Mobile (< 768px)
- Sidebar: Could be hamburger menu (not implemented)
- Content: Full width
- Document grid: 1-2 columns

## Animation Effects

1. **Page Transitions**: Fade in + slide up (0.4s)
2. **Toast Notifications**: Slide in from right (0.3s)
3. **Card Hover**: Lift + shadow (0.2s)
4. **Progress Bar**: Smooth fill (0.3s)
5. **List Items**: Stagger effect (0.05s delay each)

## Empty States

Each page has helpful empty states:

1. **Upload**: "No documents uploaded yet"
2. **Documents**: "No documents yet" + CTA button
3. **Q&A**: "Select a document" or "No questions asked"
4. **History**: "No history yet" or "No results found"
